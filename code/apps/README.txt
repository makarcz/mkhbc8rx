
Project:
	MKHBCOS, OS / runtime library and applications for MKHBC-8-Rx computer.
    ROMLIB, OS / firmware and system functions in EPROM that can be used by
    external programs.

Purpose: 
    Build firmware / OS and ROM library image for 8 kB EPROM of MKHBC-8-Rx
    hobby computer.
	Build runtime library mkhbcos.lib for applications running in MKHBCOS
    environment	on MKHBC-8-Rx computer.
	Build applications for MKHBCOS / MKHBC-8-Rx.
